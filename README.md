# 📊 Data Analytics Project

## 🔍 Overview

This project demonstrates an end-to-end data analytics workflow, covering data loading, exploratory data analysis (EDA), data cleaning, SQL querying across multiple database systems, and interactive dashboard creation.

The goal is to extract meaningful insights from raw data and present them in a clear, business-friendly format.

## 🚀 How to Run

1. **Clone the repository**

   ```bash
   git clone <your-repo-link>
   cd <project-folder>
   ```

2. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Set up database connections**

   * Update credentials for PostgreSQL, MySQL, and SQL Server in the configuration files.

4. **Run Python scripts**

   ```bash
   python main.py
   ```

5. **Open Power BI Dashboard**

   * Load the `.pbix` file in Power BI Desktop to explore the dashboard.
